O desafio foi criar o DW de Vendas clássico de Kimball no formato Star Schema no Postges e
importá-lo para o Power BI. Posteriormente resolver um conjunto de consultas na linguagem Data Analysis Expressions (DAX). No final do desafio, essas consultas serão apresentas em um painel no Power BI que deverá considerar a comunicação visual e a experiência do usuário.
